# React Portfolio

Your portfolio converted to React.js with Vite.

## Run the project

```bash
cd portfolio-react
npm run dev
```

## Build for production

```bash
npm run build
```

## Project Structure

- `src/App.jsx` - Main app component with theme management
- `src/components/` - All React components (Header, Hero, About, Skills, Experience, Contact, Footer)
- `src/App.css` - All styles from your original portfolio

## Features

- ✅ React components with hooks
- ✅ Theme toggle (light/dark mode)
- ✅ Scroll reveal animations
- ✅ Mobile responsive navigation
- ✅ Form handling
- ✅ All original styles preserved
